Petunjuk :
Ekstrak dulu Antrian-Dashboard.zip dan letakkan di Server pada folder SIMRS Khanza berada (posisi sejajar dengan webapps dll)

1. edit conf.php, isi yang sesuai dengan pengaturan conf.php yang ada di server masing-masing

2. buka DB tabel antriapotek2, antriapotek3, antripoli dan ubah di kolom status menjadi '0','1','2','3'
   buka DB tabel antriloketcetak dan tambahkan ini :
ALTER TABLE antriloketcetak 
ADD COLUMN status ENUM('menunggu','dipanggil','selesai') DEFAULT 'menunggu';

3. untuk mengatur jadwal Waktu sholat di kota wilayah masing-masing, buka file clock.js
cari :
        const response = await fetch(`https://api.aladhan.com/v1/timingsByCity/${day}-${month}-${year}?city=Malang&country=Indonesia&method=20`);

ganti Malang dengan wilayah masing-masing

4. (Opsional), buka file di main_menu/index.php cari pada bagian KYC...jika tidak punya KYC, hapus saja mulai dari :
          <a href="../kyc-library-php?nocache=<?= time() ?>" target="_blank" rel="noopener noreferrer">
            <img src="assets/img/kyc.png" alt="KYC">
            <span>KYC</span>
          </a>

5. Semoga berhasil dan aman tanpa terkendala

untuk buka : http://ip server/folder institusi (jika ada)/main_menu/ atau 
http://ip server/main_menu/
